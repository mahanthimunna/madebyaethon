Mahanthi Kannapalli Portfolio Redesign

Files included:
- index.html
- style.css
- script.js
- Redesigned project detail pages using the same filenames as the old portfolio.

How to use:
1. Backup your current GitHub portfolio folder.
2. Copy these files into the same folder where your existing images/ folder already lives.
3. Keep the images/ folder exactly as it is. This redesign uses the same image paths.
4. When your resume is ready, add it as resume.pdf in the same folder, or edit the Resume section link in index.html.
5. Commit and push to GitHub Pages.

Notes:
- The design avoids neon and uses a cinematic bronze/graphite 3D archive style.
- Existing YouTube embed links were preserved from the old pages.
- The Deadly Script and Side Scroller pages still use the placeholder/invalid video src from the old files, because no final embed link was present in the source files.
